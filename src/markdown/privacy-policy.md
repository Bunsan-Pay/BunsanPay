# プライバシーポリシー

ぶんさんPay（以下，「当社」といいます。）は，本ウェブサイト上で提供するサービス（以下，「本サービス」といいます。）における，ユーザーの個人情報の取扱いについて，以下のとおりプライバシーポリシー（以下，「本ポリシー」といいます。）を定めます。

## 第1条（個人情報）
「個人情報」とは，個人情報保護法にいう「個人情報」を指すものとし，生存する個人に関する情報であって，当該情報に含まれる氏名，生年月日，住所，電話番号，連絡先その他の記述等により特定の個人を識別できる情報（個人識別情報）を指します。  
なお、本サービスにおいては、ブロックチェーン上のウォレットアドレスやトランザクション履歴等の情報も、特定の個人を識別できる情報と紐づけられた場合には個人情報として扱われる可能性があります。  
また、ウォレットアドレス、トランザクションデータ、オンチェーンに記録されたメッセージ等は、単体では個人情報に該当しない場合であっても、他の情報と照合され個人が識別可能となる場合には「個人関連情報」または「個人情報」として取り扱われることがあります。


## 第2条（個人情報の収集方法および利用目的）
1. 当社は、ユーザーが本サービスを利用する際に、ウォレットアドレス、IPアドレス、アクセスログ等の情報を取得する場合があります。
2. 収集した情報は、以下の目的で利用します。
    1. 本サービスの提供・運営のため
    2. ユーザーからのお問い合わせに回答するため
    3. 利用規約に違反したユーザーや，不正・不当な目的でサービスを利用しようとするユーザーの特定をし，ご利用をお断りするため
    4. 上記の利用目的に付随する目的

## 第3条（外部サービスの利用とデータ共有）
当社は、サービスの提供、維持、改善およびセキュリティ確保のために、以下の第三者が提供するサービスを利用しており、これらのサービスプロバイダーに対し、必要な範囲でユーザーの情報が提供される場合があります。各サービスにおける情報の取り扱いについては、それぞれのプライバシーポリシーをご確認ください。

1. **Reown AppKit**
   - 用途: ウォレット接続機能、ブロックチェーンインタラクションの提供
   - 運営会社: Reown
   - プライバシーポリシー: [https://reown.com/privacy-policy](https://reown.com/privacy-policy)

2. **Alchemy**
   - 用途: トランザクションの取得等
   - 運営会社: Alchemy
   - プライバシーポリシー: [https://legal.alchemy.com/](https://legal.alchemy.com/)

3. **Cloudflare Workers**
   - 用途: アプリケーションのホスティング、セキュリティ対策、配信
   - 運営会社: Cloudflare, Inc.
   - プライバシーポリシー: [https://www.cloudflare.com/ja-jp/privacypolicy/](https://www.cloudflare.com/ja-jp/privacypolicy/)

4. 当社が直接取得・保存しない場合であっても、以下の情報が外部サービスまたはその内部依存先に送信される可能性があります。

- IPアドレス
- ウォレットアドレス
- 接続先チェーンID
- トランザクション署名・ブロードキャストに付随する技術情報

これらは、ブロックチェーンネットワークまたはSDKの仕様上、不可避的に送信される場合があります。


## 第4条（個人情報の第三者提供）
当社は，次に掲げる場合を除いて，あらかじめユーザーの同意を得ることなく，第三者に個人情報を提供することはありません。ただし，個人情報保護法その他の法令で認められる場合を除きます。
1. 人の生命，身体または財産の保護のために必要がある場合であって，本人の同意を得ることが困難であるとき
2. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって，本人の同意を得ることが困難であるとき
3. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって，本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき
4. 前条に定める外部サービスプロバイダーへの提供の場合

## 第5条（オンチェーンデータの取扱い）
ユーザーが任意で入力し、ブロックチェーン上に記録される情報（メッセージ、ウォレットアドレス、識別子等）は、ブロックチェーンの特性上、当社による削除・修正ができません。
ユーザーは、当該情報が公開される可能性があることを理解した上で入力するものとします。

## 第6条（プライバシーポリシーの変更）
1. 本ポリシーの内容は，法令その他本ポリシーに別段の定めのある事項を除いて，ユーザーに通知することなく，変更することができるものとします。
2. 当社が別途定める場合を除いて，変更後のプライバシーポリシーは，本ウェブサイトに掲載したときから効力を生じるものとします。
